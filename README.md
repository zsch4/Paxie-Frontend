# PAXIE: N-Pax AI Chat Assistant

## Description
PAXIE is a real-time AI chat assistant built for N-Pax website, featuring a responsive chat interface with smooth animations for messages and chat modals. It helps users interact with AI in an intuitive and engaging way.

## Features
### Guest User
- Responsive chat interface for desktop and mobile
- AI-powered real-time responses
- User-friendly input and message display

### Admin User
- Responsive admin interface
- User authentication (Login/Logout)
- View and upload documents or files

## Frontend Tech Stacks
- **Framework:** React 18
- **Styling:** Tailwind CSS
- **Build Tool:** Vite
- **Icons:** Lucide React

## Packages and Tools
- `axios` – for HTTP requests to the backend API
- `react-router-dom` – for routing (if multi-page)
- Node.js 18+ and npm 9+ recommended
